### Passos para Instalação

1. Clone o repositório: 
   ```bash
   git clone https://github.com/caiqueSanderson/at-react-fundamentals.git

2. Navegue até o diretório do projeto:
   ```bash
   cd booking-hotel

3. Instale as dependências:
    ```bash
   npm i

4. Inicie o servidor local:
   ```bash
   npm start
